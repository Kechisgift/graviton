# Changelog

## Versioning Legend

Versioning is in Animal Emoji Lexicographical Order (AELO). For example:

1. 🦙 (Alpaca)
2. 🐗 (Boar)
3. 🐈 (Cat)

... etc ...

## Tagging Cheatsheet

* create an annotated tag:
  * `git tag -as 🦙 -m "productionize graviton" && git push origin 🦙`
* get tag details:
  * `git show 🦙`
